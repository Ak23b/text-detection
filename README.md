# text-detection

- Script to detect text regions in an image

## Usage

- Before running, make sure you have the image(s) containg text are in the project folder

1. clone the repository "git clone and the link to this particular repo"

2. cd "text-detection"

3. run the script "python text_detect.py"

4. The script will depict the results of the operations. 

- The model isn't the best when it comes to text detection.
